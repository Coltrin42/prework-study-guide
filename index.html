Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds/stages modifications to the current working branch. -A adds all changes
  • git commit -m: takes everything from staging and makes permanent.
  • git pull origin main: Pull receives a branchs modifications into local environment. Origin indicates the source of the pull will be the GitHub repo. Main indicates the branch.
  • git push origin *branch*: pushes local branch to remote repository.
